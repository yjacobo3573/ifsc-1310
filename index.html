"Thoughts on Progressive Enhancement"

Progressive enhancement is a methodology that makes the content the most important part of a website and makes it available to everyone regardless of their browser. Moreover, if someone has a browser that allows for advanced code or faster internet connection then they receive an enhanced version of the website, which may include CSS and JavaScript. Progressive enhancement is important in modern web development because it makes sure that anyone that has internet access can access the content on a website. This is incredibly important because it doesn’t leave anyone behind and it doesn’t discriminate against people with slow internet connection or old browsers.
